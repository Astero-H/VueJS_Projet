# VueJS_Projet
==============

Projet VueJS - NodeServer  - LP DAWIN Alt
-----------------------------------------

**Récupérer le projet** : 
        git pull https://github.com/Astero-H/VueJS_Projet.git

**Compiler et lancer le projet** :
        npm install && node_modules/.bin/webpack --progress --hide-modules && node src/node/app.js
